# Project overview

This project is the portfolio itself: a bilingual, recruiter-first site that turns content into a product surface.

## Problem

The legacy version mixed presentation, content, and navigation logic. That made it hard to scale the site as a portfolio, a writing platform, and a proof-of-work artifact at the same time.

## Approach

I rebuilt the information architecture around a small set of canonical pages and reusable content contracts.

## Key decisions

- Treat content as structured data instead of hardcoded page copy.
- Make English the source of truth and support Vietnamese as a secondary locale.
- Keep the UI restrained so the work, not the chrome, carries the narrative.

## Results

- Cleaner case-study storytelling
- Simpler maintenance path
- Stronger recruiter scanning flow
- Better fit for future projects and blog posts
