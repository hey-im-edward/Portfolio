# Workflow nội dung MDX cho portfolio developer

MDX phù hợp với portfolio vì nó cho bạn hai thứ cùng lúc: nội dung có cấu trúc và đủ linh hoạt để viết như con người.

## Mô hình nội dung

Tôi thích chia nội dung thành các nhóm rõ ràng:

- Trang tĩnh
- Project
- Bài blog
- Site settings

Mỗi nhóm có schema riêng. Mỗi schema có field rõ ràng. Nhờ đó site ít phải đoán và dễ phát triển hơn.
