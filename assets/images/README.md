# Image Assets

Store images here for Hugo Pipes processing.

## Structure

```
assets/images/
├── hero/           # Hero images for posts
├── blog/           # Blog post images
├── icons/          # SVG icons
└── og-default.png  # Default Open Graph image
```

## Best Practices

1. Use descriptive filenames: `java-stream-api-diagram.png`
2. Store at original resolution (min 1200px for responsive)
3. Hugo will auto-generate WebP/AVIF versions
4. Always include in page bundle when possible
