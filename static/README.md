# Static Files

Place static assets here that should be copied as-is to `public/`:

- `favicon.ico`
- `favicon.svg`
- `apple-touch-icon.png`
- `site.webmanifest`
- `robots.txt` (if custom, otherwise Hugo generates)

## Recommended Files

```
static/
├── favicon.ico
├── favicon.svg
├── apple-touch-icon.png (180x180)
├── favicon-32x32.png
├── favicon-16x16.png
├── site.webmanifest
└── images/
    └── og-default.png (1200x630)
```

## Generate Favicons

Use <https://realfavicongenerator.net/> to generate all required formats.
